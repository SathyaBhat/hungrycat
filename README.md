# Hungry Cat.

The cat is hungry. Discord bot will feed it.

### Getting started

1. Install CDK

```bash
npm install aws-cdk
```e `cdk.json` file tells the CDK Toolkit how to execute your app.

2. Bootstap

```bash
npx cdk bootstrap
```

3. Create the function & deploy the code

```bash
npx cdk deploy
```
