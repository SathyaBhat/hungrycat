# Hungry Cat.

The cat is hungry. This Discord bot will provide suggestions on what to feed the hungry cat.

### Getting started

1. Install CDK

```bash
npm install aws-cdk
```e `cdk.json` file tells the CDK Toolkit how to execute your app.

2. Bootstap

```bash
npx cdk bootstrap
```

3. Create the function & deploy the code

```bash
npx cdk deploy
```
